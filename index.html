📲 Installation manuelle (sécurisée)

1

Télécharger l'APK

Cliquez sur le bouton ci-dessus pour obtenir le fichier .apk.

2

Autoriser « Sources inconnues »

Paramètres → Sécurité / Applications inconnues → Activer l'installation depuis votre navigateur ou gestionnaire de fichiers.

3

Installer l'application

Ouvrez le fichier téléchargé et validez l’installation.

4

Lancer LiveTV

Profitez du streaming directement sur votre écran !

⚠️
Installation hors Play Store – LiveTV n’est pas distribuée sur Google Play. Vous devez autoriser manuellement l’installation depuis les paramètres Android. Aucune donnée personnelle n’est collectée.
